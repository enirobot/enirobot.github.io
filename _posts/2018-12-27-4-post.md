자세한 내용은 : [node.js](https://nodejs.org/en/)

nvm(Node Version Manager)는 node.js의 버전을 관리 해준다.
여러 버전을 설치하고 바꿔가며 사용할 수 있다.

## nvm로 node.js 설치

LTS 최신 버전 설치

```bash
$ nvm install --lts
```

## 확인

```bash
$ node -v
```
